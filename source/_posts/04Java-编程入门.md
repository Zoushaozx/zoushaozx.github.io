# 数据类型

---

```
基本数据类型 描述一些具体的数字单元 
	数值型
		整型
			byte		 short    int        long 默认值 0
			-128-127  32767   21478
		浮点型 
			float double 默认值 0.0
	布尔型 boolean 默认值false
	字符型 char 默认值`u0000`
	
数据类型选择参考原则，
	描述数字首先int double；
	数据传输，文字编辑使用byte二进制处理
	处理中文使用char
	描述内存文件大小，描述表注解爱你列（主动增长） long
```

```
引用数据类型 牵扯到内存关系的使用 默认值null
	数组
  类
  接口

```

```
	Integer.MIN_VALUE 获取int最小值

```
